# Moon_Trace
一个用于web安全的工具集，目前处于信息收集制作阶段


# 使用
## 环境
需要go语言环境。第三方库使用go mod管理。

## 功能
### 子域扫描
支持：DNS数据集与ce证书透明度查询
语法： `go run moon_trace.go -u xxx.com -sub`
### 端口扫描 
支持： tcp扫描
语法： `go run moon_trace.go -u www.xxx.com -port`
